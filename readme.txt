== Description ==

Adds special pages used to customize the single post view for Groups and Jobs.

Groups and Jobs are custom post type (CPT) with predefined content. You can not edit these post's views with a page builder.
This plugin creates special pages that can be edited with a page builder and places predefined content (a group or a job) inside this page using a shortcode.

= Key Features =

- Ability to customize the single group view of the "Ultimate Member - Groups" plugin with a page builder.
- Ability to customize the single job view of the "JobBoardWP" plugin with a page builder.
